When would you want to use a remote repository rather than keeping all your work local?
  When collaborating with others.
  
  
Why might you want to always pull changes manually rather than having Git automatically stay up-to-date with your remote repository?
  Changes can cause conflicts. If you're in the middle of creating a new feature, it's best to chose when to pull and incorporate other people's changes so that you can deal with conflicts at times when you're features are mature.
